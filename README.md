## Android Application | Body Mass Index
Android BMI calculator supporting localization and color-coded results based on BMI categories. Handled full development process including UI/UX design and complete implementation.

### Project Overview
This project was conducted as part of the **Mobile Application Development** course, a core subject in the third year of study. The project involved developing an **Android application for Body Mass Index (BMI) calculation**.  

The application supports **language switching (localization)** and displays **color-coded results** that accurately represent predefined BMI categories.

I was responsible for **all aspects of the project**, including application design, UI/UX design, and full implementation and coding, ensuring both functional accuracy and a user-friendly experience.  

*(Developed using Android Studio)*
